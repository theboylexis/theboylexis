## Projects

### [Bus Passenger Seat Detection](https://github.com/theboylexis/trotro-seat-detection)
An ESP32-based seat occupancy detection and passenger accounting system for Ghanaian public buses. Proof of concept for tamper-resistant revenue tracking in commercial minibuses.

**Highlights:** GPIO seat sensing · pull-down resistor circuit · Serial Monitor output · scalable to 14 seats · physical prototype in progress

`ESP32` `Arduino IDE` `C++` `Embedded Systems` `IoT`

### [Smart Doc API](https://github.com/theboylexis/smart-doc-api)
A production-ready document intelligence API. Upload PDFs, DOCX, or TXT files and get AI-powered analysis via OpenAI — built with real-world backend engineering practices.

**Highlights:** JWT auth · BullMQ job queues · Socket.io real-time updates · Redis caching · Webhooks · Rate limiting · Docker · CI/CD · 32 integration tests

`Node.js` `Express` `PostgreSQL` `Prisma` `Redis` `BullMQ` `OpenAI` `Docker`

[Live API](https://smart-doc-api-production.up.railway.app) · [Swagger Docs](https://smart-doc-api-production.up.railway.app/api-docs)
